# ChatCat
A ChatBot overlay for answering UofA students questions about programs, classes, and more!
## Packages
- backend: Backend API using Java Spring 
- frontend: Frontend built with React served with Flask
- model: LLM extended with RAG using LightRAG library.
## Setup
-Run the rag.py file in the Model folder to start the backend
-Have node.js installed on your machine
-in a terminal go to the directory that the project is located at and change directory into the frontend folder
-type in "npm start" and this will auto open the front end which will connect to the already running rag.py file
